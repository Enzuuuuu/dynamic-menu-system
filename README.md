# 🍽️ Cardápio Digital Customizável

Sistema de cardápio digital desenvolvido para atender diferentes estabelecimentos utilizando uma única base de código altamente configurável.

## ✨ Objetivo

Permitir a criação rápida de cardápios digitais personalizados para restaurantes, lanchonetes, cafeterias e outros negócios, reduzindo o tempo necessário para implantação em novos clientes.

## 🚀 Funcionalidades

- Exibição de categorias e produtos
- Carrinho de compras
- Banner principal personalizado
- Informações do estabelecimento
- Design responsivo
- Configuração simplificada para novos clientes
- Estrutura modular e reutilizável

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript ES6+
- Tailwind CSS

## 📂 Estrutura de Personalização

O sistema foi projetado para que a adaptação para novos clientes exija apenas alterações em arquivos específicos de configuração, como:

- Dados do restaurante
- Produtos
- Categorias
- Imagens
- Informações de contato
